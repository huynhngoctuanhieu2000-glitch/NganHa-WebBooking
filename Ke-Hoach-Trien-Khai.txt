───────────────────────────────────

  • Các phần độc lập có thể làm song song
  • i18n tích hợp ngay từ đầu (không để cuối)
  • Hiệu ứng menu lật sách → tích hợp khi có Service Cards


2. THỨ TỰ CÔNG VIỆC (11 BƯỚC)
──────────────────────────────

╔═══════════════════════════════════════════════════════════╗
║  PHASE 1A: NỀN TẢNG (Làm đầu tiên)                     ║
╚═══════════════════════════════════════════════════════════╝

  Bước 1: Init Project + Design System
  ─────────────────────────────────────
    → Tạo project Next.js (TypeScript, Tailwind, App Router)
    → Cài: framer-motion, lucide-react, react-pageflip, 
           @supabase/supabase-js
    → Cấu hình Tailwind theme (gold/dark palette)
    → Setup Supabase clients
    → Setup constants.ts, .env.local template

  Bước 2: i18n Infrastructure
  ────────────────────────────
    → Cấu trúc routing: app/[locale]/
    → Dictionaries 5 ngôn ngữ: vi, en, cn, jp, kr
    → Middleware chuyển ngữ

  Bước 3: Root Layout + Header
  ─────────────────────────────
    → Root layout (fonts, metadata, SEO)
    → Header: sticky nav, logo, language switcher, mobile menu
    → Files: Header.tsx / Header.logic.ts / Header.i18n.ts


╔═══════════════════════════════════════════════════════════╗
║  PHASE 1B: LANDING PAGE (Sau khi có nền tảng)           ║
╚═══════════════════════════════════════════════════════════╝

  Bước 4: Hero Section
  ─────────────────────
    → Fullscreen background video/image
    → 2 chi nhánh CTAs (link Google Maps)
    → Entrance animations (Framer Motion)

  Bước 5: Service Cards + MENU LẬT SÁCH 📖
  ──────────────────────────────────────────
    → Thư viện: react-pageflip (StPageFlip wrapper)
    → Mỗi "trang sách" = 1 gói dịch vụ (hình + giá)
    → Hiệu ứng 3D page-flip khi vuốt/click
    → Bìa sách: tông vàng gold sang trọng
    → Mobile: single page, vuốt ngang
    → Desktop: double page, click cạnh
    → Fetch dữ liệu từ Supabase

  Bước 6: Branch Info
  ────────────────────
    → 2 chi nhánh: 11 Ngô Đức Kế & 6B Thi Sách
    → Google Maps link
    → Card layout sang trọng

  Bước 7: Promotions Section
  ───────────────────────────
    → Chương trình khuyến mãi
    → Nút "Xem chi tiết" + "Đặt lịch ngay"

  Bước 8: Footer + Floating Widgets
  ──────────────────────────────────
    → Footer: info, social links, copyright
    → Floating buttons: Zalo, FB Messenger, WhatsApp, 
      Line, KakaoTalk, WeChat, Hotline
    → Fixed position, luôn hiển thị


╔═══════════════════════════════════════════════════════════╗
║  PHASE 1C: BOOKING FLOW (Core Feature)                   ║
╚═══════════════════════════════════════════════════════════╝

  Bước 9: Booking Form
  ─────────────────────
    → Chọn dịch vụ, ngày giờ, chi nhánh
    → Điền: tên, SĐT, email
    → Validation + submit to Supabase
    → Files: BookingForm.tsx / .logic.ts / .i18n.ts

  Bước 10: VietQR Payment
  ────────────────────────
    → Tạo mã QR từ VietQR Quick Link (không cần API key)
    → URL: img.vietqr.io/image/{bank}-{account}-{template}.png
    → Hiển thị đúng số tiền + ghi chú
    → Status: PENDING_PAYMENT → CONFIRMED


╔═══════════════════════════════════════════════════════════╗
║  PHASE 1D: POLISH (Hoàn thiện)                           ║
╚═══════════════════════════════════════════════════════════╝

  Bước 11: Responsive + SEO + Performance
  ────────────────────────────────────────
    → Test: 375px, 768px, 1024px, 1440px
    → Touch targets >= 44px (mobile)
    → SEO meta tags
    → Performance optimization
